<h1 align="center">NarraLeaf Project</h1>

<h4 align="center">视觉小说新定义</h3>

<p align="center"><a href="./README.md">English</a> | 简体中文</p>

NarraLeaf Project 是一个现代视觉小说游戏引擎，提供多种解决方案。从灵活集成到统一开发，它帮助释放创作者的创造力。

## 为什么选择 NarraLeaf？

- **更轻量** - 无需重型渲染库，利用原生 Web 技术减少对低端设备和移动设备的压力。

- **更简单** - 不想写代码？集成式的可视化开发环境帮助您专注于释放创造力。

- **更专业** - CI/CD 流水线、版本控制系统、代码检查工具。与您的团队构建真正的现代化应用程序。

此外，NarraLeaf...

- 一款应用，多个平台。以零障碍的方式为 Web 和桌面构建游戏。  
- 强大的生态系统。使用完整的 TypeScript 生态系统构建条件逻辑、可复用组件和动态叙事。  
- 高度可扩展。NarraLeaf 中的每个概念都简单、优雅且直观，让您的项目随着需求一同扩展。

## 解决方案

### [NarraLeaf Studio](https://github.com/NarraLeaf/NarraLeaf-Studio) - 一体化视觉小说 IDE

零代码、一体化视觉小说 IDE。

包含所有基本开发工作流的编辑器。不需要编码，随时开始释放您的创造力。

### [NarraLeaf](https://github.com/NarraLeaf/NarraLeaf) – 桌面端完整解决方案

Node.js 库和 CLI 工具，用于处理开发工作流的每个部分。

像创建 React 应用一样轻松构建跨平台桌面视觉小说。NarraLeaf 为您处理其他所有事情；比如游戏流程、测试和打包。

### [NarraLeaf-React](https://github.com/NarraLeaf/narraleaf-react) - 内嵌式 VN 播放器解决方案

专为 React 构建的轻量级前端视觉小说播放器。

NarraLeaf-React 为引人入胜的故事叙述提供您所需的一切：叙事、界面和性能。通过其简单、直观的 API，将您的舞台带到任何 Web 平台。
